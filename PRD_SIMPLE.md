# 售货机质检管理系统 - 简易PRD

## 1. 产品概述

本系统用于展示售货机大货质检的检测结果，支持质检人员查看、筛选质检记录及详情，提升质检工作效率。

## 2. 页面功能

### 2.1 质检报告-售货机检测页

- **功能**:
  - 搜索：支持按SKU、SN码、设备型号搜索
  - 筛选：支持按检测结果（全部/通过/不通过）筛选
  - 展示：SKU、SN码、设备型号、检测结果、操作列
  - 分页：每页5条记录，支持翻页

### 2.2 全部售货机检测结果页

- **功能**:
  - 搜索：支持按SKU、PO、质检批次号、SN码、设备型号、质检员搜索
  - 筛选：支持按检测结果（全部/通过/不通过）筛选
  - 展示：SKU、PO、质检批次号、实际质检时间、质检员、设备型号、SN码、检测结果、操作列
  - 分页：每页5条记录，支持翻页
  - 导航：提供返回"质检报告-售货机检测"页的链接

### 2.3 售货机检测详情页

- **功能**:
  - 基础信息：SKU、PO、质检批次号、实际质检时间、质检员、设备型号、SN码
  - 检测明细：检测项（POS检测）、检测内容、检测结果
  - 导航：提供返回列表页的链接

## 3. 核心流程

1. 质检人员完成其他质检项后，通过移动端检测售货机
2. 整改完成后提交最终检测结果
3. 系统展示最终结果，通过数量回写至质检报告结论

## 4. 数据字段

### 质检记录

| 字段     | 说明     | 示例                   |
| ------ | ------ | -------------------- |
| SKU    | 商品编码   | PTG\_337L6R4K        |
| PO     | 采购订单号  | PO26031014772205     |
| 质检批次号  | 质检批次编号 | INSP\_PB44559562-5-1 |
| SN码    | 设备序列号  | SN2024010001         |
| 设备型号   | 售货机型号  | VM-1000              |
| 实际质检时间 | 检测时间   | 2024-03-15 10:30:25  |
| 质检员    | 检测人员   | 张三                   |
| 检测结果   | 通过/不通过 | 通过                   |

### 检测明细

| 字段   | 说明         | 示例           |
| ---- | ---------- | ------------ |
| 检测项  | 检测项目       | POS检测        |
| 检测内容 | 检测描述       | 检测POS机功能是否正常 |
| 检测结果 | 通过/不通过/待检测 | 通过           |

## 5. 技术实现

- **框架**: React + TypeScript
- **样式**: TailwindCSS
- **图标**: Lucide React
- **路由**: React Router
- **构建**: Vite

## 6. 页面关系

```
质检报告-售货机检测页 ↔ 全部售货机检测结果页
        ↓
售货机检测详情页
```
