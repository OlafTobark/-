# 售货机质检管理系统 PRD

## 1. 产品概览

本系统旨在为售货机大货质检提供管理功能，支持质检记录的查看、筛选和详情查看，帮助质检人员高效管理售货机的质检流程。

## 2. 核心功能

### 2.1 功能模块

| 模块名称       | 功能描述                             |
| ---------- | -------------------------------- |
| 质检报告-售货机检测 | 展示本次质检的售货机检测的结果，支持搜索和筛选，提供查看详情功能 |
| 全部售货机检测结果  | 展示所有售货机的检测结果，包含完整字段信息            | 
| 售货机检测详情    | 展示所有售货机的检测结果，包括基础信息和检测明细         | 

### 2.2 页面详情

| 页面名称        | 模块名称 | 功能描述                                  |
| ----------- | ---- | ------------------------------------- |
| 质检报告-售货机检测页 | 搜索功能 | 支持按SKU、SN码、设备型号搜索                     |
| 质检报告-售货机检测页 | 筛选功能 | 支持按检测结果（全部/通过/不通过）筛选                  | 
| 质检报告-售货机检测页 | 数据展示 | 展示SKU、SN码、设备型号、检测结果、操作列               | 
| 质检报告-售货机检测页 | 分页功能 | 每页显示5条记录，支持翻页                         | 
| 质检报告-售货机检测页 | 导航功能 | 提供跳转到全部售货机检测结果页面的链接                   |
| 全部售货机检测结果页  | 搜索功能 | 支持按SKU、PO、质检批次号、SN码、设备型号、质检员搜索              | 
| 全部售货机检测结果页  | 筛选功能 | 支持按检测结果（全部/通过/不通过）筛选                  | 
| 全部售货机检测结果页  | 数据展示 | 展示SKU、PO、质检批次号、实际质检时间、质检员、设备型号、SN码、检测结果、操作列 | 
| 全部售货机检测结果页  | 分页功能 | 每页显示5条记录，支持翻页                         | 
| 全部售货机检测结果页  | 导航功能 | 提供返回质检报告-售货机检测页面的链接                   | 
| 售货机检测详情页    | 基础信息 | 展示SKU、PO、实际质检时间、质检员、设备型号、SN码          | 
| 售货机检测详情页    | 检测明细 | 展示检测项、检测内容、检测结果                       | 
| 售货机检测详情页    | 导航功能 | 提供返回列表页面的链接                           | 

## 3. 核心流程

1. 质检同事将其他质检项质检完成后，通过质检的开始使用移动端检测售货机检测内容 
2. 当质检通过的这部分货物，售货机检测内容整改完成后可以提交最终结果 
3. 系统展示的就是最终的结果，提交后的通过数量，回写到质检报告结论里面的通过数量

## 4. 用户界面设计

### 4.1 设计风格

- 风格：现代B端管理系统风格
- 主色调：蓝色系 
- 辅助色：绿色（通过）、红色（不通过）
- 按钮风格：圆角按钮，有 hover 效果 
- 字体：系统默认字体 
- 布局：响应式布局，适配不同屏幕尺寸

## 5. 数据结构

### 5.1 质检记录

| 字段名              | 数据类型   | 描述                        |
| ---------------- | ------ | ------------------------- |
| id               | number | 记录ID                      |
| sku              | string | 商品SKU，格式：PTG337L6R4K    | 
| po               | string | 采购订单号，格式：PO26031014772205 |
| snCode           | string | 设备SN码，移动端录入               | 
| deviceModel      | string | 设备型号，移动端录入                |
| inspectionTime   | string | 实际质检时间                    | 
| inspector        | string | 质检员                       | 
| inspectionResult | string | 检测结果（passed/failed），移动端录入 | 
| details          | array  | 检测明细，移动端录入                | 

### 5.2 检测明细

| 字段名     | 数据类型   | 描述                          |
| ------- | ------ | --------------------------- |
| id      | number | 明细ID                        |
| item    | string | 检测项                         | 
| content | string | 检测内容                        | 
| result  | string | 检测结果（passed/failed/pending） | 

## 6. 技术实现

### 6.1 技术栈

- 前端框架：React + TypeScript
- 样式：TailwindCSS
- 图标：Lucide React 
- 路由：React Router 
- 构建工具：Vite

### 6.2 项目结构

```
/src
  /components
  /pages
    QualityInspectionList.tsx     # 质检报告-售货机检测页面
    AllMachineInspectionList.tsx  # 全部售货机检测结果页面
    QualityInspectionDetail.tsx   # 售货机检测详情页面
  App.tsx                          # 应用入口（路由配置）
  main.tsx                         # React入口
  index.css                        # 全局样式
```
